The JavaScript apply() Method

Com o método apply (), você pode escrever um método que pode ser usado em objetos diferentes.

The Difference Between call() and apply()

A diferença é: O método call () recebe argumentos separadamente.
O método apply () aceita argumentos como uma matriz.

OBS: O método apply () é muito útil se você quiser usar uma matriz em vez de uma lista de argumentos.
