Array.forEach()

O método forEach () chama uma função (uma função de retorno de chamada) uma vez para cada elemento da matriz.

O método map () cria uma nova matriz executando uma função em cada elemento da matriz. O método map () não executa a função para elementos de matriz sem valores.
