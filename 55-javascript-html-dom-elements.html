Finding HTML Elements

Muitas vezes, com JavaScript, você quer manipular elementos HTML. Para fazer isso, você precisa encontrar os elementos primeiro. Existem várias maneiras de fazer isso:

  • Encontrando elementos HTML por id
  • Encontrando elementos HTML pelo nome da tag
  • Encontrando elementos HTML por nome de classe
  • Encontrando elementos HTML por seletores CSS
  • Encontrando elementos HTML por coleções de objetos HTML
