JavaScript Objects

Objetos JavaScript são escritos com chaves. As propriedades do objeto são gravadas como pares nome: valor, separados por vírgulas.

O objeto (person) no exemplo acima possui 4 propriedades: firstName, lastName, age e eyeColor.
