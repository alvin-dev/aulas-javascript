Um Boolean JavaScript representa um dos dois valores: verdadeiro ou falso. (true or false)

Muitas vezes, na programação, você precisará de um tipo de dados que só pode ter um dos dois valores, como

  • YES / NO
  • ON / OFF
  • TRUE / FALSE

Para isso, o JavaScript possui um tipo de dados booleano. Só pode ter os valores verdadeiros ou falsos.
