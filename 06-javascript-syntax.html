JavaScript Numbers

O número pode ser escrito com ou sem decimais.

Strings são textos escritos entre aspas duplas ou simples:

Ex:
"John Doe"
'John Doe'

Variáveis JavaScript

Em uma linguagem de programação, variáveis são usadas para armazenar valores de dados.
JavaScript usa a palavra-chave var para declarar variáveis.
Um sinal de igual é usado para atribuir valores a variáveis. Neste exemplo, x é definido como uma variável. Então, x é atribuído (dado) o valor 6:

JavaScript Operadores

JavaScript usa operadores aritméticos (+ - * /) para calcular valores:
Ex:
(5 + 6) * 10

JavaScript usa um operador de atribuição (=) para atribuir valores a variáveis:
Ex:
var x, y;
x = 5;
y = 6;
