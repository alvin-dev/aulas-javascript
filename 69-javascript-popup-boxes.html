O JavaScript possui três tipos de caixas pop-up: caixa de alerta, caixa de confirmação e caixa de prompts.

Alert Box

Uma caixa de alerta é freqüentemente usada se você quiser garantir que as informações cheguem ao usuário. Quando uma caixa de alerta aparecer, o usuário terá que clicar em "OK" para prosseguir.

Confirm Box

Uma caixa de confirmação é freqüentemente usada se você quiser que o usuário verifique ou aceite alguma coisa. Quando uma caixa de confirmação aparece, o usuário terá que clicar em "OK" ou "Cancelar" para prosseguir.

Prompt Box

Uma caixa de aviso é frequentemente usada se você quiser que o usuário insira um valor antes de entrar em uma página. Quando uma caixa de aviso aparecer, o usuário terá que clicar em "OK" ou "Cancelar" para prosseguir após inserir um valor de entrada.

window.prompt("sometext","defaultText");

Se o usuário clicar em "OK", a caixa retornará o valor de entrada. Se o usuário clicar em "Cancelar", a caixa retornará nulo.

Line Breaks

Para exibir as quebras de linha dentro de uma caixa pop-up, use uma barra invertida seguida do caractere n.

alert("Hello\nHow are you?");
