Object Types (Blueprints) (Classes)

Às vezes precisamos de um "plano" para criar muitos objetos do mesmo "tipo".
A maneira de criar um "tipo de objeto" é usar uma função construtora de objetos.
No exemplo acima, a função Person () é uma função de construtor de objeto.

Objects of the same type are created by calling the constructor function with the new keyword:

The this Keyword

Em JavaScript, a coisa chamada "this" é o objeto que "possui" o código.

O valor "this", quando usado em um objeto, é o próprio objeto.

Em uma função de construtor, "this" não tem um valor. É um substituto para o novo objeto. O valor "this" se tornará o novo objeto quando um novo objeto for criado.
