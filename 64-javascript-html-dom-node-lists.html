The HTML DOM NodeList Object

Um objeto NodeList é uma lista (coleção) de nós extraídos de um documento. Um objeto NodeList é quase o mesmo que um objeto HTMLCollection.

Alguns (mais antigos) navegadores retornam um objeto NodeList em vez de um HTMLCollection para métodos como getElementsByClassName (). Todos os navegadores retornam um objeto NodeList para a propriedade childNodes. A maioria dos navegadores retorna um objeto NodeList para o método querySelectorAll (). Um HTMLCollection (capítulo anterior) é uma coleção de elementos HTML. Um NodeList é uma coleção de nós de documentos. Uma coleção NodeList e uma coleção HTML é basicamente a mesma coisa.

Um HTMLCollection (capítulo anterior) é uma coleção de elementos HTML. Um NodeList é uma coleção de nós de documentos. Uma coleção NodeList e uma coleção HTML é basicamente a mesma coisa.

Tanto um objeto HTMLCollection quanto um objeto NodeList são uma lista (coleção) semelhante a um array de objetos. Ambas possuem uma propriedade length definindo o número de itens na lista (coleção).

Ambos fornecem um índice (0, 1, 2, 3, 4, ...) para acessar cada item como uma matriz. Itens HTMLCollection podem ser acessados pelo seu nome, número de identificação ou número de índice. Os itens de NodeList só podem ser acessados pelo seu número de índice.
