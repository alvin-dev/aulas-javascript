Invoking a JavaScript Function

O código dentro de uma função não é executado quando a função é definida. O código dentro de uma função é executado quando a função é invocada. É comum usar o termo "chamar uma função" em vez de "invocar uma função".

Também é comum dizer "chamar uma função", "iniciar uma função" ou "executar uma função". Neste tutorial, usaremos invoke, porque uma função JavaScript pode ser chamada sem ser chamada.
