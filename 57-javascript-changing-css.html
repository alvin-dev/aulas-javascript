O HTML DOM permite que o JavaScript altere o estilo dos elementos HTML.

Changing HTML Style

Para alterar o estilo de um elemento HTML, use esta sintaxe:

document.getElementById(id).style.property = new style

Using Events

O HTML DOM permite que você execute código quando ocorre um evento. Eventos são gerados pelo navegador quando "coisas acontecem" para elementos HTML:

  • Um elemento é clicado
  • A página foi carregada
  • Campos de entrada são alterados
