O objeto de documento HTML DOM é o proprietário de todos os outros objetos em sua página da web.

The HTML DOM Document Object

O objeto do documento representa sua página da web. Se você quiser acessar qualquer elemento em uma página HTML, você sempre começará a acessar o objeto de documento. Abaixo estão alguns exemplos de como você pode usar o objeto de documento para acessar e manipular o HTML.

Finding HTML Elements

  • document.getElementById(id) - Encontre um elemento por id de elemento
  • document.getElementsByTagName(name) - Encontrar elementos pelo nome da tag
  • document.getElementsByClassName(name) - Encontrar elementos por nome de classe

Changing HTML Elements

  • element.innerHTML = new html content - Alterar o HTML interno de um elemento
  • element.attribute = new value - Alterar o valor do atributo de um elemento HTML
  • element.setAttribute(attribute, value) - Alterar o valor do atributo de um elemento HTML
  • element.style.property = new style - Alterar o estilo de um elemento HTML
