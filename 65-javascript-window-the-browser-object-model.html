O Browser Object Model (BOM) permite que o JavaScript "fale com" o navegador.

The Window Object

O window object é suportado por todos os navegadores. Representa a janela do navegador. Todos os objetos, funções e variáveis globais do JavaScript tornam-se automaticamente membros do objeto da janela. Variáveis globais são propriedades do objeto da janela.
