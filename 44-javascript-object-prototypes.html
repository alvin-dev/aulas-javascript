Herança de protótipo

Todos os objetos JavaScript herdam propriedades e métodos de um protótipo:

  • Os objetos de "date" herdam de Date.prototype
  • Objetos de "array" herdam de Array.prototype
  • Objetos de "person" herdam de Person.prototype

O Object.prototype está no topo da cadeia de herança do protótipo:
Objetos de data, objetos Array e objetos Person herdam de Object.prototype.
